# 项目提案：Avoid-Nav Beijing (进京避让导航)

## 1. 产品定位
为在北京行驶的外地车主提供“合法合规、精准避让”的专业导航辅助。基于 11w+ 动态更新的监控数据，实时生成规避抓拍点位的安全路径。

## 2. 核心用户交互 (UX)
1. **环境感知**：开屏即展北京全城 11w+ 监控点位（海量点 Canvas 图层），颜色区分风险等级（红/蓝/紫）。
2. **条件配置**：用户选择“合规现状”（无证/六环外证/已办证）。
3. **安全导航**：输入目的地 -> 系统自动搜索路径 500m 内危险点 -> 动态注入高德 API `avoidpolygons` -> 生成规避路线。
4. **实时播报**：靠近监控点时触发语音提醒。

## 3. 商业模式
- **工具变现**：提供增值避让服务（离线地图、精准点位纠偏）。
- **流量引导**：进京证代办咨询、保险及周边服务。

## 4. 数据更新策略
- 支持通过外部脚本定期更新 `data.json`。
- 本地预处理脚本自动精炼数据并刷新空间索引。
